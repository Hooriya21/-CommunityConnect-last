# CommunityConnect - Hackathon Project

## Quick Start
1. Open `public/index.html` in browser
2. Or run: `cd server && npm install && node server.js`
3. Visit `http://localhost:3000`

## Features
- Real-time neighborhood skill exchange
- Interactive map with live markers
- Kindness points system
- Demo mode for presentations

## Tech Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Node.js + Express
- Database: Firebase (simulated in demo)
- Maps: Leaflet.js
- Styling: Tailwind CSS
